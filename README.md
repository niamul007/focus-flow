# FocusFlow 🌊
A professional Full-Stack Relational Task Management System built with a layered architecture.

## 🏛️ System Architecture
This project uses a **Layered Monorepo** structure to separate concerns and ensure scalability:
- **Client:** React (Vite) - *[Upcoming]*
- **Server:** Node.js & Express (ES Modules)
- **Database:** PostgreSQL (Neon.tech)
- **Validation:** Zod (Type-safe schemas)

## 📂 Folder Structure
- `/server`: Express API with Layered Architecture (Controllers, Services, Routes).
- `/client`: React Frontend interface.
